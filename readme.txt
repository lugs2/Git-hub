El propósito de este proyecto es demostrar su capacidad para recopilar, trabajar y limpiar un conjunto de datos. El objetivo es preparar los datos ordenada que puede ser utilizado para un análisis posterior. 
Se le evaluará por sus pares en una serie de preguntas sí / no relacionados con el proyecto. Usted tendrá que presentar:
 1) un conjunto como se describe a continuación los datos ordenados,
 2) un enlace a un repositorio de Github con su guión para realizar el análisis
3) un libro de códigos que describe las variables, los datos y las transformaciones o el trabajo que ha realizado para limpiar los datos llamados CodeBook.md.

 También debe incluir una README.md en el repo con tus scripts. Esta repo explica cómo todos los scripts funcionan y cómo están conectados.

Una de las áreas más interesantes en toda la ciencia de datos en este momento es la computación portátil - véase, por ejemplo este artículo. Empresas como Fitbit, Nike, y la quijada para arriba están compitiendo para desarrollar los algoritmos más avanzados para atraer a nuevos usuarios. Los datos vinculados a partir del sitio web del curso representan los datos obtenidos de los acelerómetros del teléfono inteligente Samsung Galaxy S. Una descripción completa se encuentra disponible en el sitio donde se obtuvieron los datos:



http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones 

Here are the data for the project: 

https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip 


Usted debe crear un script llamado R run_analysis.R que hace lo siguiente.

1. fusiona la formación y la prueba de conjuntos para crear un conjunto de datos .
2. Los extractos sólo las mediciones en la media y la desviación estándar para cada medición.
3. Utiliza nombres descriptivos de actividad para nombrar las actividades en el conjunto de datos
4. Etiquetas apropiadamente el conjunto de datos con los nombres de variables descriptivas.
5. A partir de los datos establecidos en el paso 4 , crea establecidos con el promedio de cada variable para cada actividad y cada sujeto un segundo dato ordenado, independientes.
